# प्रोजेक्ट योजना (Planning)

**उद्देश्य:** Git और GitHub को पूरी तरह हिंदी में सीखना।  
**Checkpoint-1 का लक्ष्य:**  
- फोल्डर बनाना, नोट्स तैयार करना और Git को 'main' ब्रांच के साथ शुरू करना।

### कदम (Steps):
1. प्रोजेक्ट के लिए एक साफ़-सुथरा फोल्डर बनाना।
2. ज़रूरी दस्तावेज़ (Documentation) फाइलें बनाना।
3. Git की ग्लोबल सेटिंग में 'main' को डिफ़ॉल्ट ब्रांच सेट करना।
4. Git को चालू (Initialize) करना।
